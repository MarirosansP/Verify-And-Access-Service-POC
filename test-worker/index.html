CloudFlare Worker Simulator

This page simulates what a CloudFlare Worker would do: call the Verify & Access API to initiate a verification, then redirect the user to the verification page and poll for results.

Ready. Fill in your worker key and click a button.\n

Curl Commands

Alternatively, test from your terminal:

# Step 1 — Initiate verification curl -X POST http://localhost:3001/api/worker/initiate \ -H "Content-Type: application/json" \ -H "X-Worker-Key: wk_YOUR_KEY_HERE" \ -d '{"challenge":"age_over_18","callbackUrl":"http://localhost:3004"}' # Step 2 — Open verifyUrl in browser (from step 1 response) # Step 3 — Poll for result curl http://localhost:3001/api/worker/status/SESSION_ID_HERE \ -H "X-Worker-Key: wk_YOUR_KEY_HERE"
